### はじめに

アプリ紹介の記事を書こうとするたびに思っていたこと。

「スクショ、面倒くさいな」

Windows 11の `Win+Shift+S` は範囲指定できて便利だけど、撮った画像はまずクリップボードに入るだけ。自動保存機能もあるにはあるが、設定をONにする必要があり、保存先は `ピクチャ\スクリーンショット` に固定で変えられない。アプリごとに日付フォルダで整理、なんてことはできないのだ。

かといってネットに転がっているスクリーンショットツールは機能が多すぎて覚えることが多かったり、動作が重かったりと、なんとなく使う気になれなかった。

**「PrintScreenを押したら、日付フォルダに勝手に保存されるだけでいい」**

そんなシンプルなものを作ってしまいました。

---

### SS Captureとは

**SS Capture**は、ホットキー一発でアクティブウィンドウや全画面のスクリーンショットをPNGファイルとして保存するWindowsツールです。

Python製で、EXEにまとめてあるのでPythonのインストールは不要です。

---

### 主な機能

**① ホットキーでキャプチャ（アクティブウィンドウ）**

デフォルトは `Print Screen` キー。今前面にあるウィンドウだけをきれいにキャプチャします。SS Capture自身がアクティブになっていても、直前に使っていたウィンドウを自動的に取得するので、「アプリの画面じゃなく自分が撮れた」という事故もありません。

**② 全画面ホットキー**

`Shift + Print Screen` で全画面も撮れます。マルチモニター環境にも対応。

**③ 保存先フォルダを自由に設定**

デフォルトは `ピクチャ\SSCapture\` 配下ですが、任意のフォルダに変更できます。日付ごとのサブフォルダが自動で作られ、連番でPNG保存されます。設定は次回起動時も引き継がれます。

```
ピクチャ\SSCapture\
  └── 20260628\
        ├── 0001.png
        ├── 0002.png
        └── 0003.png
```

**④ クリップボード監視**

「クリップボードの画像を自動保存」をONにすると、`Win+Shift+S` で範囲選択したスクリーンショットも、ファイルに自動保存されます。外部ツールとの組み合わせが自由自在。

**⑤ クリップボードへの同時コピー**

キャプチャと同時にクリップボードにも画像を送れます。そのままメモ帳やSlackに貼り付けられます。

**⑥ トレイ常駐・スタートアップ対応**

×ボタンを押してもトレイに格納されて常駐します。スタートアップ登録もアプリ内から一発。

---

### 使い方

1. `SSCapture.exe` を起動
2. ホットキーを押すだけ

- **アクティブウィンドウを保存** → 設定したホットキーを押す（デフォルト：`Print Screen`）
- **全画面を保存** → 設定したホットキーを押す（デフォルト：`Shift + Print Screen`）
- **範囲指定して保存** → `Win+Shift+S` で範囲選択 → クリップボード監視ONで自動保存

任意のフォルダ / 日付 / 連番.png に保存されます。それだけです。

---

### こだわったポイント

このツールで一番こだわったのは、**3つの保存方法を1つのツールでカバーすること**。

- **アクティブウィンドウ** → ホットキー一発で前面のウィンドウだけ保存
- **全画面** → 別ホットキーでデスクトップ全体を保存
- **範囲指定（Win+Shift+S）** → クリップボード監視ONにしておけば自動でファイル保存

どの方法で撮っても、設定した同じフォルダに日付ごとに連番で整理される。ツールを使い分ける必要がなく、撮ったスクショが一箇所にまとまるのが地味に便利です。

機能の数ではなく「刺さるかどうか」で作ったツールです。

- 毎回範囲指定が面倒な人
- 撮ったスクショを日付フォルダで整理したい人
- シンプルで軽いキャプチャツールを探している人

そんな方に使ってもらえたら嬉しいです。
